Pocket G-Code Generator (GUI)

A professional Tkinter application for CNC pocketing operations

Developed by Amir Mobasheraghdam

A lightweight Tkinter app to generate safe, layered pocketing G-code for CNC routers. Supports single-hole entry and batch CSV import, with built-in validation, concentric passes, optional finish passes, live statistics, and WinPC-NC-friendly output.
Safety First: Always do a dry run above the material and verify toolpaths before cutting.

üì¶ Installation

Requirements

  • Windows / macOS / Linux with Python 3.9+
  • Tkinter (usually included with Python installations)

Setup Instructions

# Clone the repository git clone https://github.com/AmirMobasheraghdam/pocket-gcode-generator.git cd pocket-gcode-generator # (Optional) Create virtual environment python -m venv .venv source .venv/bin/activate # Windows: .venv\Scripts\activate # Run the application python main.py
Note for Linux users: On some distributions, Tkinter may need to be installed separately:
sudo apt-get install python3-tk

üöÄ Usage

Single Pocket Generation

Enter pocket coordinates (X, Y)
Set pocket diameter and depth
Configure machining parameters (stepdown, stepover, feeds, spindle speed, dwell)
Optionally enable finish pass
Click "Generate G-Code" to create toolpath

Batch Processing

Prepare CSV file with columns: X, Y, Diameter, Depth
Click "Import CSV" and select your file
Review imported pockets in the list
Generate G-code for all pockets

Profile Management

Save your machining parameters as profiles for future use
Load existing profiles to quickly apply settings

üìÑ Output Format

The generator produces WinPC-NC compatible G-code including:

  • Proper initialization (G21 G90 G17 G94 G54)
  • Spindle control (M3/M5)
  • Correct arc movements (G3 with IJ notation)
  • Safe retraction between pockets
  • Comprehensive logging to logs/ directory
